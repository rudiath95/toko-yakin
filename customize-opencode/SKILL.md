# customize-opencode

## Description
Use when editing or creating opencode's own configuration files: `opencode.json`, `opencode.jsonc`, files under `.opencode/`, or files under `~/.config/opencode/`. Also use when creating or fixing opencode agents, subagents, skills, plugins, MCP servers, or permission rules.

## When to use
- Configuring opencode itself (not the user's application code)
- Creating or fixing opencode agents, subagents, or skills
- Setting up MCP servers or permission rules for opencode

## Usage
Load this skill when the task matches any of the above scenarios. The skill provides domain-specific instructions and workflows for opencode configuration.
